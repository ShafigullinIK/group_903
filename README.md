Это сторонний проект на GitHub аккаунте.


1. Делаем (fork) интересующего нас репозитория.
2. Мы делаем git clone для нашей версии этого репозитория.
3. Создаём ветку с предлогаемыми изменениями.
4. Призводим все изменения только в этой ветке.
5. Отправляем эти изменения на свой аккаунт (push).
6. В окне на GitHub появляется возможность отправить pull request с изменениями.
