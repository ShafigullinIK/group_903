Это какой-то проект на github аккаунте

1. fork интересующего нас репозитория
2. git clone для моей версии репозитория
3. Создаю ветку с предлагаемыми изменениями
4. Произвожу все изменения только в этой ветке
5. Отправляю эти изменения на свой аккаунт через git push
6. В окне на github появляется кнопка отправки pull request для предложения моих изменений автору оригинального репозитория аккаунта автора
